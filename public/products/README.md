# Product Images Directory

## How to Add Product Images

1. **Save your product images in this folder**: `public/products/`
2. **Use descriptive filenames**: `milan-top-liquid-soap.jpg`
3. **Supported formats**: JPG, PNG, WebP
4. **Recommended size**: 400x400px or larger (square format works best)

## Usage in Code

When you add an image here, reference it in your product data as:
```javascript
image: '/products/your-image-name.png'  // or .jpg, .webp
```

## Image Optimization Tips

- Keep file sizes under 500KB for faster loading
- Use JPG for photos, PNG for graphics with transparency
- Square aspect ratio (1:1) works best for product cards
